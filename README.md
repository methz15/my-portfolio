# Personal Portfolio Website

This is my personal developer portfolio built using Next.js and Tailwind CSS.  
The website showcases my projects, technical skills, and contact information.

## Tech Stack
- Next.js
- Tailwind CSS
- JavaScript
- React
- Git & GitHub

## Features
- Responsive modern UI
- Project showcase section
- Skills and technologies section
- Contact information
- Optimized performance and SEO

## Live Website


## Installation

Clone the repository:

git clone 

Install dependencies:

npm install

Run the development server:

npm run dev

## Deployment
This project is deployed using Vercel.
